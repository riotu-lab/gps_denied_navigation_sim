# Taif DEM Model Launch Guide\n\n## Overview\n\nThis guide explains how to launch the `taif_dem` model in PX4 Gazebo simulation. The Taif DEM (Digital Elevation Model) provides realistic terrain for GPS-denied navigation testing in a mountainous Saudi Arabian environment.\n\n## Available DEM Models\n\nThe PX4 workspace includes two Taif terrain models:\n\n| Model | Location | World File | Description |\n|-------|----------|------------|-------------|\n| `taif_dem` | `PX4-Autopilot/Tools/simulation/gz/models/taif_dem/` | `taif_world.sdf` | Main Taif terrain model |\n| `taif1_dem` | `PX4-Autopilot/Tools/simulation/gz/models/taif1_dem/` | `taif1_world.sdf` | Alternative Taif terrain |\n\n## Model Structure\n\n### taif_dem Configuration\n```xml\n<!-- model.sdf -->\n<model name=\"taif_terrain\">\n  <pose>0 0 0 0 0 0</pose>\n  <static>true</static>\n  <geometry>\n    <mesh>\n      <uri>media/taif1.dae</uri>\n      <scale>1 1 1</scale>\n    </mesh>\n  </geometry>\n</model>\n```\n\n### Geographic Coordinates\n- **Center**: 21.2713°N, 40.3460°E (Taif, Saudi Arabia)\n- **Altitude**: 1874.6 meters above sea level\n- **Coverage**: Mountainous terrain of Taif region\n\n## Launch Methods\n\n### Method 1: Using make with specific world\n```bash\n# Navigate to PX4 directory\ncd PX4-Autopilot\n\n# Launch with taif_world (includes taif_dem model)\nmake px4_sitl gz_x500 PX4_GZ_WORLD=taif_world\n\n# Alternative: Launch with taif1_world\nmake px4_sitl gz_x500 PX4_GZ_WORLD=taif1_world\n```\n\n### Method 2: Using specific vehicle + world combinations\n```bash\n# With stereo camera and 3D LiDAR\nmake px4_sitl gz_x500_stereo_cam_3d_lidar PX4_GZ_WORLD=taif_world\n\n# With twin stereo cameras and twin Velodyne LiDAR\nmake px4_sitl gz_x500_twin_stereo_twin_velodyne PX4_GZ_WORLD=taif_world\n\n# With mono camera and 3D LiDAR\nmake px4_sitl gz_x500_mono_cam_3d_lidar PX4_GZ_WORLD=taif_world\n```\n\n### Method 3: Direct Gazebo launch\n```bash\n# Set environment variables\nexport GZ_SIM_RESOURCE_PATH=$GZ_SIM_RESOURCE_PATH:$(pwd)/Tools/simulation/gz/models\n\n# Launch Gazebo with taif world\ngz sim -r Tools/simulation/gz/worlds/taif_world.sdf\n```\n\n### Method 4: Custom launch script\n```bash\n#!/bin/bash\n# taif_launch.sh\n\ncd PX4-Autopilot\n\n# Set model path\nexport GZ_SIM_RESOURCE_PATH=$GZ_SIM_RESOURCE_PATH:$(pwd)/Tools/simulation/gz/models\n\n# Launch with specific parameters\nmake px4_sitl gz_x500_stereo_cam_3d_lidar PX4_GZ_WORLD=taif_world \\\n     PX4_SIM_SPEED_FACTOR=1 \\\n     PX4_GZ_MODEL_POSE=\"0,0,1900,0,0,0\"\n```\n\n## Environment Variables\n\n### Key Environment Variables\n```bash\n# World selection\nexport PX4_GZ_WORLD=taif_world\n\n# Model resource path\nexport GZ_SIM_RESOURCE_PATH=$GZ_SIM_RESOURCE_PATH:/path/to/models\n\n# Vehicle spawn position (X,Y,Z,Roll,Pitch,Yaw)\nexport PX4_GZ_MODEL_POSE=\"0,0,1900,0,0,0\"  # 1900m altitude for terrain clearance\n\n# Simulation speed\nexport PX4_SIM_SPEED_FACTOR=1.0\n```\n\n## Vehicle Configuration for Terrain\n\n### Recommended Vehicles for Taif DEM\n\n1. **x500_stereo_cam_3d_lidar**: Best for SLAM and navigation\n2. **x500_twin_stereo_twin_velodyne**: Dual sensor redundancy\n3. **x500_mono_cam_3d_lidar**: Lightweight option\n4. **x500_vision**: Visual-based navigation\n\n### Spawn Position Considerations\n\n```bash\n# Safe spawn positions (avoid terrain collision)\n# Center area (flat): X=0, Y=0, Z=1900\n# Northern area: X=100, Y=200, Z=1920\n# Southern valley: X=-50, Y=-100, Z=1880\n\n# Example spawn command\nmake px4_sitl gz_x500_stereo_cam_3d_lidar PX4_GZ_WORLD=taif_world PX4_GZ_MODEL_POSE=\"0,0,1900,0,0,0\"\n```\n\n## World Configuration Details\n\n### Physics Settings\n```xml\n<physics type=\"ode\">\n  <max_step_size>0.004</max_step_size>\n  <real_time_factor>1.0</real_time_factor>\n  <real_time_update_rate>250</real_time_update_rate>\n</physics>\n```\n\n### Lighting Configuration\n```xml\n<light name='sunUTC' type='directional'>\n  <direction>0.001 0.625 -0.78</direction>\n  <diffuse>0.904 0.904 0.904 1</diffuse>\n</light>\n```\n\n### Spherical Coordinates\n```xml\n<spherical_coordinates>\n  <surface_model>EARTH_WGS84</surface_model>\n  <world_frame_orientation>ENU</world_frame_orientation>\n  <latitude_deg>21.2713</latitude_deg>\n  <longitude_deg>40.3460</longitude_deg>\n  <altitude>1874.6</altitude>\n</spherical_coordinates>\n```\n\n## Troubleshooting\n\n### Common Issues\n\n1. **Model not found**\n   ```bash\n   # Check model path\n   echo $GZ_SIM_RESOURCE_PATH\n   \n   # Add PX4 models to path\n   export GZ_SIM_RESOURCE_PATH=$GZ_SIM_RESOURCE_PATH:$(pwd)/Tools/simulation/gz/models\n   ```\n\n2. **Vehicle spawns underground**\n   ```bash\n   # Increase spawn altitude\n   export PX4_GZ_MODEL_POSE=\"0,0,1950,0,0,0\"\n   ```\n\n3. **Terrain not loading**\n   ```bash\n   # Check mesh files exist\n   ls Tools/simulation/gz/models/taif_dem/media/\n   \n   # Verify .dae mesh file is present\n   file Tools/simulation/gz/models/taif_dem/media/taif1.dae\n   ```\n\n4. **Performance issues**\n   ```bash\n   # Reduce simulation speed\n   export PX4_SIM_SPEED_FACTOR=0.5\n   \n   # Run headless (no GUI)\n   export HEADLESS=1\n   ```\n\n### Debug Commands\n\n```bash\n# Check Gazebo model loading\ngz model --list\n\n# Verify world loading\ngz world --list\n\n# Check mesh file integrity\ngdalinfo Tools/simulation/gz/models/taif_dem/media/\n\n# Test minimal launch\ngz sim empty.sdf\n```\n\n## Advanced Configuration\n\n### Custom Mission Planning\n\n```bash\n# Launch with QGroundControl integration\nmake px4_sitl gz_x500_stereo_cam_3d_lidar PX4_GZ_WORLD=taif_world &\n\n# Start QGroundControl for mission planning\nqgroundcontrol\n```\n\n### Multi-vehicle Setup\n\n```bash\n# Vehicle 1\nPX4_SYS_AUTOSTART=4001 PX4_GZ_MODEL_POSE=\"0,0,1900,0,0,0\" make px4_sitl gz_x500\n\n# Vehicle 2 (in separate terminal)\nPX4_SYS_AUTOSTART=4001 PX4_GZ_MODEL_POSE=\"100,100,1900,0,0,0\" make px4_sitl gz_x500 PX4_INSTANCE=1\n```\n\n### Sensor Configuration\n\n```bash\n# Enable specific sensors\nexport PX4_GZ_MODEL=x500_stereo_cam_3d_lidar\nexport PX4_GZ_WORLD=taif_world\n\n# Launch with sensor logging\nmake px4_sitl gz_$PX4_GZ_MODEL PX4_GZ_WORLD=$PX4_GZ_WORLD\n```\n\n## Performance Optimization\n\n### System Requirements\n- **RAM**: Minimum 8GB, Recommended 16GB+\n- **GPU**: Dedicated graphics card recommended\n- **CPU**: Multi-core processor (4+ cores)\n\n### Optimization Settings\n\n```bash\n# Reduce physics update rate\nexport PX4_GZ_PHYSICS_STEP=0.01  # Default: 0.004\n\n# Disable shadows for better performance\nexport PX4_GZ_SHADOWS=false\n\n# Reduce camera resolution\nexport PX4_GZ_CAM_RES=\"640x480\"  # Default: 1920x1080\n```\n\n## Integration with Navigation Systems\n\n### SLAM Configuration\n```bash\n# Launch with RTAB-Map SLAM\nmake px4_sitl gz_x500_stereo_cam_3d_lidar PX4_GZ_WORLD=taif_world &\nroslaunch rtabmap_ros px4_stereo_slam.launch\n```\n\n### Visual-Inertial Odometry\n```bash\n# Launch with VIO pipeline\nmake px4_sitl gz_x500_vision PX4_GZ_WORLD=taif_world &\nroslaunch px4_vio vio_estimation.launch\n```\n\n## Related Documentation\n\n- [PX4 Gazebo Simulation](https://docs.px4.io/main/en/sim_gazebo_gz/)\n- [PX4 SITL
